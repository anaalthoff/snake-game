# Snake
## 🐍 Descrição do Jogo
Este projeto implementa um jogo **Snake** usando a biblioteca **Lanterna**.
O objetivo é simples: controla a serpente, come comida para crescer e evita bater nas paredes ou no próprio corpo.

O jogo inclui spawn aleatório de comida, crescimento da serpente, colisões e ecrã de Game Over com opção de reiniciar ou sair.

Para um **README** mais detalhado carregar [aqui](docs/README.md).
